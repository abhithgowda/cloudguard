# CloudGuard — AWS Cost & Security Governance Engine

A production-grade serverless system that continuously monitors an AWS account for cost anomalies, security violations, and zombie resources — then auto-remediates and reports.

> Full documentation will be added in STEP 22. Architecture diagram in `docs/architecture.md`.

## Stack
- Python 3.12 · Terraform ≥ 1.5 · AWS Lambda · Step Functions · EventBridge · DynamoDB · S3 · SNS/SES · KMS · IAM · CloudWatch · GitHub Actions
